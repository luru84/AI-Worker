# Contributing

## Scope
- このリポジトリは `AI-Worker` の運用ルール、役割定義、セキュリティ方針を管理します
- 実装コードは兄弟リポジトリで管理します

## Before Opening a PR
1. `work/` に機密情報や一時作業が含まれていないことを確認する
2. `SECURITY.md` と `DATA_HANDLING.md` に反していないことを確認する
3. 変更理由を `README.md`, `docs/`, `agent/` のどこに反映するか明確にする

## PR Expectations
- 変更目的を短く書く
- 新しい agent を増やす時は責務境界を書く
- セキュリティや公開範囲に影響する変更は明示する
- 実装 repo 固有の話はこの repo に持ち込まない
