# Agent Workspace

ここには親 Codex と子 Codex の恒久ルールだけを置きます。日々の会話や一時メモは `work/` 側で管理します。

- `assistant-agent.md`: ユーザーとの会話窓口
- `orchestrator.md`: 親 Codex の責務
- `researcher.md`: 調査と解析の進め方
- `security-agent.md`: 秘密情報、個人情報、権限境界の確認
- `repo-implementer.md`: 実装リポジトリでの作業ルール
- `github-operator.md`: Git/GitHub 操作の分離
- `context-curator.md`: 再利用知識の整理
- `reviewer.md`: レビュー観点
- `qa-runner.md`: 受け入れ確認と再現確認
- `token-guard.md`: トークン節約ルール
- `decisions.md`: 複数作業をまたいで参照したい判断ログ
